## 📱 نظرة عامة

Abu-Zahra Admin هو نظام إدارة أجهزة متكامل يتيح لك التحكم الكامل في الأجهزة عن بعد عبر:
- 🖥️ **لوحة تحكم ويب** - واجهة احترافية لإدارة الأجهزة
- 🤖 **بوت تيلجرام** - تحكم عبر البوت باللغة العربية
- 📱 **تطبيق Android** - خفيف وفعال على الجهاز المستهدف

## 🛠️ التقنيات المستخدمة

### السيرفر
- **Python 3.11+** - Flask + WebSocket
- **SQLite** - قاعدة بيانات
- **Telegram Bot API** - بوت تيلجرام
- **Nginx** - خادم الويب
- **Docker** - النشر السحابي

### تطبيق Android
- **Kotlin** - لغة البرمجة
- **Coroutines** - العمليات غير المتزامنة
- **OkHttp** - الاتصال بالسيرفر
- **WebSocket** - البث المباشر
- **Foreground Services** - الخدمات الخلفية

## 📦 التثبيت

### متطلبات السيرفر
```bash
# Python 3.11+
pip install -r server/requirements.txt

# تشغيل السيرفر
python server/server.py
```

### إعداد البوت
1. إنشاء بوت من [@BotFather](https://t.me/BotFather)
2. إضافة التوكن في ملف الإعدادات
3. تشغيل السيرفر

### ربط الجهاز
1. تحميل التطبيق على الجهاز
2. الحصول على كود الربط من البوت
3. إدخال الكود في التطبيق
